# 🍽️ Recipe Basket

Recipe Basket is a full-stack recipe recommendation web application built using Flask. Users can search for recipes, save favorites, manage shopping lists, review recipes, update their profile, and request ingredient orders. An admin panel allows management of users, reviews, favorites, shopping lists, and ingredient orders.

## ✨ Features

- User Registration & Login
- Forgot Password
- Profile Management
- Profile Picture Upload
- Recipe Search using Spoonacular API
- Recipe Details
- Favorite Recipes
- Shopping List
- Ingredient Order Request
- Recipe Reviews & Ratings
- Admin Dashboard
- User Management
- Review Management
- Favorite Management
- Shopping Management
- Order Management
